Welcome to the Game of life

This is a 0 players game, the rules are:

  • Any live cell with fewer than two live neighbours dies, as if by underpopulation.
  • Any live cell with two or three live neighbours lives on to the next generation.
  • Any live cell with more than three live neighbours dies, as if by overpopulation.
  • Any dead cell with exactly three live neighbours becomes a live cell, as if by reproduction.

Select the "initial cells" in the next grid

For start press 's' key or press 'Start Game Of Life' button below.
Press 'q' for hide or show the 'Next Generation' button.

Bienvenido al Juego de la Vida

Este es un juego para 0 jugadores, las reglas son:

  • Cualquier célula con menos de dos células vecinas muere, por soledad.
  • Cualquier célula con dos o tres células vecinas vivas vive para la siguiente generación.
  • Cualquier célula con más de tres células vecinas vivas muere, por sobrepoblación.
  • Cualquier célula con exactamente tres células vecinas vivas se convirte en una célula viva, por reproducción.

Selecciona las "células iniciales" en la siguiente cuadrícula

Para iniciar presiona la tecla 's' o presiona el botón 'Start Game Of Life' de abajo
Presiona 'q' para esconder o mostrar el botón 'Next Generation'.
